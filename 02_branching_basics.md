# create a new branch

Create a new branch called `from _itc`.

```python
git checkout -b from_itc

# output
Switched to a new branch 'from_itc'
```

To see a list of your branches and which branch you're on, enter `git branch`.

```python
git branch

# output
* from_itc
  master
```
